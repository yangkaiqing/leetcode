## 查找算法

**二分查找**

```python
# 二分查找的输入必须是有序的
# 复杂度，log n 这里算法复杂度是指随着输入的增加，算法操作次数的增加
def binary_search(nums, target):
    left  = 0
    right = len(nums) - 1
    while left <= right:
        mid = (left+right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left  = mid + 1
        else:
            right = mid - 1
    return -1
```

制作花束m需要的最少天数

```python
#天数肯定落在花开的最小值和最大值之间，因此相当于寻找左侧边界
def minDays(self, bloomDay: List[int], m: int, k: int) -> int:
        if m*k > len(bloomDay):
            return -1
        def canmake(day):
            flower  = 0
            bouquet = 0
            for x, elem in enumerate(bloomDay):
                if elem <= day:
                    flower += 1
                    if flower == k:
                        bouquet += 1
                        flower  =  0
                else:
                    flower = 0
                if bouquet >= m:
                    break
            return bouquet >= m
        left  = min(bloomDay)
        right = max(bloomDay)
        while left < right:
            mid = (left+right) // 2
            if canmake(mid):
                right = mid
            else:
                left = mid + 1
        return left
```
