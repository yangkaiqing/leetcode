## 查找算法

**二分查找**

```python
# 二分查找的输入必须是有序的
# 复杂度，log n 这里算法复杂度是指随着输入的增加，算法操作次数的增加
def binary_search(nums, target):
    left  = 0
    right = len(nums) - 1
    while left <= right:
        mid = (left+right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left  = mid + 1
        else:
            right = mid - 1
    return -1
```
