## 回溯算法

电话号码的字母数字组合

```
def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        d = {2:"abc", 3:"def", 4:"ghi",
                5:"jkl", 6:"mno", 7:"pqrs",
                8:"tuv", 9:"wxyz"}
        def backtrack(res, s, digits, index):
            # 递归终止条件
            if len(s) == len(digits):
                res.append(s)
                return
            s_temp = d[int(digits[index])]
            for c in s_temp:
                #选择当前值
                s += c
                backtrack(res, s, digits, index+1)
                #回溯
                s = s[:-1]
            return
        res = []
        backtrack(res, "", digits, 0)
        return res
```
